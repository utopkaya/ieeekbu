UMUT TOPKAYA

Endüstri Mühendisliği

Merhabalar! Ben Umut. Karabük Üniversitesi'nde Endüstri Mühendisliği 2.sınıf öğrencisiyim. Ortaokul öğretmenimin vasıtasıyla yazılıma olan ilgi ve merakım o yıldan itibaren devam etmekte. Lise son sınıfta İspanya - Sevilla'da Güneş Enerjisi Sistemleri üzerinde staj deneyimim oldu. Lisede çeşitli Arduino destekli robotik projeler geliştirdim. Python, React, Java teknolojileri ile çeşitli ve basit düzeyde uygulamalar geliştirdim. Geliştirdiğim uygulamaları Github profilimde paylaşmaktayım. Günümüz dünyasının gereksinimi olan yapay zeka & veri bilimi alanında ilerleme katetmek için Python düzeyimi geliştiriyorum.
